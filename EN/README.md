# Kubernetes course by modules (EN)

This directory contains the complete Kubernetes learning module in English,
from foundations to professional operations and a final project.

Link to the Spanish README: [README ES](../ES/README.md)

## Course summary

- Module 0: Foundations, DevEx, and reproducible environment
- Module 1: Containers, Docker, Podman, and Compose
- Module 2: Why Kubernetes exists
- Module 3: First cluster and kubectl
- Module 4: Kubernetes mental model
- Module 5: Pods and basic objects
- Module 6: Workloads
- Module 7: Networking
- Module 8: Configuration, secrets, and storage
- Module 9: Automated testing for Kubernetes
- Module 10: Application delivery
- Module 11: Security
- Module 12: Operations, observability, and reliability with Grafana LGTM
- Module 13: Cloud native patterns
- Module 14: Extending Kubernetes
- Module 15: Professionalization by role
- Module 16: Final roadmap project
- Module 17: GitOps and continuous delivery with Argo CD
