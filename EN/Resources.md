## Ruta recomendada según objective

For **empezar bien**, usaría this orden:

1. **Kubernetes Docs + Learn Kubernetes Basics**
2. **Introduction to Kubernetes of Linux Foundation / edX**
3. **Play with Kubernetes or Killercoda** for practicar without install nada
4. **Kubernetes in Action, Second Edition** como libro principal
5. **KodeKloud** if quieres labs guiados and preparación seria
6. **CKAD** if eres developer
7. **CKA** if quieres administración, plataforma, SRE or DevOps
8. **CKS** only after of tener good base and haber aprobado CKA
For **developer**, priorizaría:

- Kubernetes in Action
- Kubernetes Patterns
- CKAD
- KodeKloud CKAD
- Learnk8s Blog for internals
For **DevOps / SRE / Platform Engineer**, priorizaría:

- Kubernetes: Up and Running
- Kubernetes Best Practices
- Cloud Native DevOps with Kubernetes
- CKA
- Killer.sh
- Learnk8s workshops if tienes presupuesto
For **security**, priorizaría:

- CKA first
- CKS after
- Kubernetes Best Practices
- Documentación oficial
- Labs of Killercoda / Killer.sh
A nota importante: evitaría PDFs sueltos of libros comerciales que aparecen in Google. For libros of OR’Reilly, Manning u otras editoriales, better ir to the editorial, Amazon, VitalSource, OR’Reilly Learning or bibliotecas legales. Kubernetes cambia bastante and also muchos PDFs not oficiales pueden estar desactualizados or not ser legales.
