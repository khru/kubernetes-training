# Curso de Kubernetes por modulos (ES)

Este directorio contiene el modulo completo de aprendizaje de Kubernetes en espanol,
desde fundamentos hasta operacion profesional y proyecto final.

Enlace al README en ingles: [README EN](../EN/README.md)

## Resumen del curso

- Modulo 0: Fundamentos, DevEx y entorno reproducible
- Modulo 1: Contenedores, Docker, Podman y Compose
- Modulo 2: Por qué aparece Kubernetes
- Modulo 3: Primer cluster y kubectl
- Modulo 4: Modelo mental de Kubernetes
- Modulo 5: Pods y objetos básicos
- Modulo 6: Workloads
- Modulo 7: Networking
- Modulo 8: Configuración, secretos y almacenamiento
- Modulo 9: Testing automatizado de Kubernetes
- Modulo 10: Delivery de aplicaciones
- Modulo 11: Seguridad
- Modulo 12: Operación, observabilidad y fiabilidad con Grafana LGTM
- Modulo 13: Patrones cloud native
- Modulo 14: Extensión de Kubernetes
- Modulo 15: Profesionalización por rol
- Modulo 16: Proyecto final del roadmap
- Modulo 17: GitOps y delivery continuo con Argo CD.md
